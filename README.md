# Jarrod the Human

A Slack bot that collects feature requests through natural conversation and creates Linear tickets.

## Setup

1. Import this repo into Replit
2. Add these secrets in Replit:
   - SLACK_BOT_TOKEN
   - SLACK_SIGNING_SECRET  
   - SLACK_APP_TOKEN
   - ANTHROPIC_API_KEY
   - LINEAR_API_KEY
3. Click Run!

## Features

- Conversational feature request collection (no forms!)
- Automatic Linear ticket creation in KDE Product Triage
- Daily summary posted to #well-do-it-live at 5pm CDT
- Works in #kde-product-requests channel and DMs
